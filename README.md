# CustomReportMaker

A custom report generation web app built with Python Flask, pandas, HTML, CSS, and JavaScript.

## Features
- Dynamic column selection
- Filtering and sorting
- Summary cards
- Quick insights
- Growth rate analysis
- Grade and status generation
- Custom formula support
- Large dataset support

## Dataset Scale
Tested with a college-scale dataset of 2200+ students and 33,000 rows.

## Tech Stack
- Python
- Flask
- pandas
- NumPy
- HTML
- CSS
- JavaScript

## How to Run
1. Create and activate a virtual environment
2. Install dependencies
3. Run the Flask app
4. Open the local URL in browser

## Install Dependencies
```bash
pip install flask pandas numpy
